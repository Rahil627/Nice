Nice is a lightweight framework for designing static blogs & websites. It is completely open source & written in the Haxe programming language.

Simple, yet powerful.

To write posts in Nice, you use raw HTML by default. Although it is a bit more difficult for beginners, it gives a lot more power to the developer and decreases compilation times. Do you prefer Markdown? No problem. Markdown is supported out of the box.

Lightweight.

Nice weighs in at under 140 kb.

Free.

Nice doesn't cost you a dime. In fact, all of the source is available on Github.
